m2e-code-quality by bimargulies

This project builds Eclipse plugins that connect some other Eclipse plugins with their corresponding Maven plugins. If you've gone to a lot of trouble to set up just the Checkstyle, PMD or Findbugs configuration you want in Maven, and you import your Maven projects into Eclipse with m2e, you want to see the same warnings in your Eclipse editors that the maven build will complain of.

At this time, the three plugins supported are Checkstyle, PMD and Findbugs.

There is a Google Group to facilitiate communications about the use and development of the plugins: m2e-code-quality Google Group

The current snapshot/qualifier versions are available on the update site, with newer versions replacing older ones.

The most recent snapshot was at 21 July 2012, containing version 1.0.0.201207211839.

Download

To install this on Eclipse, use the 1.0.0 update-site. I'm testing this with Eclipse 3.6, 4.2 and m2e 1.0, 1.1.

As usual on github, you can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/m2e-code-quality/m2e-code-quality

get the source code on GitHub : m2e-code-quality/m2e-code-quality
